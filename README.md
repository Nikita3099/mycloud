# My Cloud (Дипломный проект)

## Что это
Веб-приложение “облачное хранилище”:
- регистрация/аутентификация пользователей;
- админка для управления пользователями и их правами;
- управление файлами: загрузка, список, удаление, переименование, изменение комментария, просмотр/скачивание, генерация обезличенной ссылки для внешнего скачивания.

## Стек
- Backend: Python + Django + Django REST Framework, БД PostgreSQL
- Frontend: React + Redux Toolkit + React Router.
- Хранение файлов: локальный диск сервера в `MEDIA_ROOT`.

## Репозиторий
- `mycloud/` — Django-приложение (API + отдача SPA).
- `frontend/` — React-приложение.
- `requirements.txt` — зависимости Python.

## Быстрый запуск (локально)

### 1) Установить зависимости Python
1. Убедитесь, что установлен Python 3.10+.
2. Создайте виртуальное окружение.
3. Установите зависимости:
   - `pip install -r requirements.txt`

### 2) Собрать фронтенд
1. `cd frontend`
2. `npm install`
3. `npm run build`

### 3) Поднять базу и применить миграции
1. `cd mycloud`
2. `python manage.py migrate`

При миграции создаётся пользователь `admin` с правами администратора:
- логин: `admin`
- пароль: `Admin1!`

### 4) Запуск сервера
- `python manage.py runserver`

Откройте в браузере:
`http://127.0.0.1:8000/`
